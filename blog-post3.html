JavaScript Temelleri

27 Temmuz 2024 Yazar: Admin

JavaScript, web geliştirmenin vazgeçilmez bir parçasıdır. Bu yazıda, JavaScript'in temel kavramlarını ve web sayfalarınızı nasıl interaktif hale getirebileceğinizi öğreneceksiniz.

1. Değişkenler ve Veri Tipleri

JavaScript'te değişken tanımlamak için let, const veya var kullanılır. Temel veri tipleri arasında string, number, boolean, null ve undefined bulunur.


let name = "Ahmet";
const age = 30;
var isStudent = false;

2. Fonksiyonlar

Fonksiyonlar, kodu yeniden kullanılabilir bloklara ayırmanızı sağlar. İşte bir örnek:


function greet(name) {
    return `Merhaba, ${name}!`;
}
console.log(greet("Ayşe")); // "Merhaba, Ayşe!" yazdırır

3. Koşullu İfadeler

if-else ifadeleri, kodunuzun belirli koşullara göre farklı davranmasını sağlar:


let hour = new Date().getHours();
if (hour < 12) {
    console.log("Günaydın!");
} else if (hour < 18) {
    console.log("İyi günler!");
} else {
    console.log("İyi akşamlar!");
}

4. Döngüler

Döngüler, bir işlemi tekrar tekrar yapmanızı sağlar. En yaygın döngü türleri for ve while'dır:


for (let i = 0; i < 5; i++) {
    console.log(i);
}

let j = 0;
while (j < 5) {
    console.log(j);
    j++;
}

5. DOM Manipülasyonu

JavaScript ile HTML elementlerini seçebilir, değiştirebilir ve yeni elementler ekleyebilirsiniz:


let title = document.getElementById("title");
title.innerHTML = "Yeni Başlık";

let newParagraph = document.createElement("p");
newParagraph.textContent = "Bu yeni bir paragraf.";
document.body.appendChild(newParagraph);

Bu temel kavramları öğrendikten sonra, JavaScript ile web sayfalarınızı daha interaktif ve dinamik hale getirebilirsiniz.
